# Sigil App Plugin Source

This folder hosts the Grafana app plugin UI for Sigil.

- `app/`: root routing and page composition.
- `pages/`: user-facing pages for conversations, completions, traces, and settings.
- `api/`: typed client contracts for Sigil query APIs.
- `plugin/`: plugin bootstrap and extension wiring.
- `services/`, `shared/`, `state/`, `stores/`: supporting runtime layers.
