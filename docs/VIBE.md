# Sigil Vibe

## Voice

- Cast, trace, reveal.
- Short, sharp, technical language.
- Emphasize observability after LLM actions.

## Product Framing

- A prompt is a spell.
- Sigil helps you inspect what happened: tokens, outputs, actions, references, and traces.
- UI should prioritize chronology, causality, and cross-linking between completion artifacts and trace spans.
